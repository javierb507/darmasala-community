# Core Design System: Atma Suddhi Premium

## Overview
This document defines the fixed core UI/UX design for the Atma Suddhi Yoga School Management System. This design is considered the stable baseline and should be preserved in all future iterations.

## Visual Identity
*   **Theme Name**: Deep Purple & Enchanted Gold (Premium Edition)
*   **Primary Colors**:
    *   `--primary-color`: #4a148c (Deep Purple) - Sidebar and primary headers.
    *   `--accent-color`: #ffd700 (Gold) - Active states, highlights, and secondary accents.
    *   `--bg-gradient`: Linear gradient from deep purple to a lighter secondary purple.
*   **Typography**:
    *   `Inter` or `Roboto` for body text (readability).
    *   `Playfair Display` or similar serif for headers (elegance).
*   **Aesthetics**: Glassmorphism, subtle micro-animations, and soft shadow containers.

## Structural Core
1.  **Unified Sidebar**: A sticky left sidebar with high-contrast icons and labels in gold/white.
2.  **Dashboard Layout**: Modular card-based layout with clean summaries (Alumnos, Calendario, Yogaterapia, Economía).
3.  **Responsive Design**: Mobile-first approach, ensuring the sidebar collapses or transforms into a hamburger menu on smaller screens.
4.  **Premium Modals**: Full-screen or centered glass-morphism modals for data entry (asistencia, pagos).

## Modification Policy
*   **Locked Elements**: The layout structure, navigation flow, and overall "Premium" feel are locked.
*   **Allowed Changes**: Minor adjustments to color tones, font-weights, or padding to improve accessibility are permitted, but only if they harmonize with the existing core.
